Image Size Test Page

  • Image are loaded directly into the page from a storage account.
  • Calls to load the images are made from the clients web browser, not the backend.
  • The storage account holds both the html page as well as the images.
  • To view the image loading times, hit F12 and use the network analyzer.
  • The image urls have a random querystring appended to keep them from being cached locally
  • The size at which the images are rendered on the page can be changed with the slider
